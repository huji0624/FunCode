处理错误

程序出错了怎么办？

首先恭喜你通过了上一课的考验，这是一个相对简单的题目，但也有可能有的同学因为一些粗心的输入错误而得不到正确的结果，这时候会蹦出一个出错的窗口，告诉你哪一行程序发生了错误。比如你在字符串的后面多写了一个s: 

print "hello"s

那么你在运行的时候会得到这样的错误信息: 

File "", line 1
print "hello"s
             ^
SyntaxError: invalid syntax

这个出错信息告诉你，你在"line 1"（第一行）有一个语法错误，在字符串后面多写了一个s，计算机不能理解这个s，于是就告诉你出错啦。

在以后的程序中你可能还会遇到其他类型的错误，出错了并不可怕，利用出错信息，认真阅读程序，总是能够修复这些错误的，因为计算机是很听话很可靠的，所以你基本上可以这样认为，计算机不会出错，错误都是由于人们编写了错误的程序引起的。
